## Purpose
The goal of this project is to a do a data analysis project with continuous integration on GitHub.

## Video

## Items Included
1. `lib.py`, `test_lib.py`
2. `main.py`, `test_main.py`
3. `Jupyter Notebook`

## Steps
1. Use the ruff template provided by Professor Gift. Modifiy below items:
    1. Adding Python packages and specify versions in requirements 
2. Create lib file to include functions that will be called in main.py file.
3. Create main file to do data analysis, including descriptive statistics and graphs. 
4. Create Jupyter Notebook to show descriptive and graphs
5. Automatically save the graphs that are generated
6. Update the test files for lib and main to ensure the functions are working

## Summary of the Dataset
The dataset 'imdb_top_1000.csv' is downloaded from Kaggle https://www.kaggle.com/datasets/juanarchidonaahijado/imdb-top-1000 . The dataset contains 16 columns and 1,000 rows.
